## 问题
[LeetCode 0206. Reverse Linked List](https://leetcode.com/problems/reverse-linked-list/)
反转链表，思路并不难，难的是怎么最简洁地写出代码

```python
class Solution:
    def reverseList(self, head: ListNode) -> ListNode:
        cur, prev = head, None
        while cur:
            cur.next, prev, cur = prev,  cur, cur.next
        return prev
```
