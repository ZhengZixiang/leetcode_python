## 问题
[LeetCode 0120](https://leetcode.com/problems/triangle/)

求解三角形的最小路径和

## 思路
一个很自然的想法就是自顶向下递归求解（或者回溯/深搜）
```python
# O(n*n/2) space, top-down
def minimumTotal(self, triangle):
    if not triangle:
        return 0
    res = [[0 for i in range(len(row))] for row in triangle]
    res[0][0] = triangle[0][0]
    for i in range(1, len(triangle)):
        for j in range(len(triangle[i])):
            if j == 0:
                res[i][j] = res[i-1][j] + triangle[i][j]
            elif j == len(triangle[i])-1:
                res[i][j] = res[i-1][j-1] + triangle[i][j]
            else:
                res[i][j] = min(res[i-1][j-1], res[i-1][j]) + triangle[i][j]
    return min(res[-1])
    
# Modify the original triangle, top-down
def minimumTotal(self, triangle):
    if not triangle:
        return 0
    for i in range(1, len(triangle)):
        for j in range(len(triangle[i])):
            if j == 0:
                triangle[i][j] += triangle[i-1][j]
            elif j == len(triangle[i])-1:
                triangle[i][j] += triangle[i-1][j-1]
            else:
                triangle[i][j] += min(triangle[i-1][j-1], triangle[i-1][j])
    return min(triangle[-1])
```
贪心做法，显然容易陷入局部最优。

DP则是自底向上的思路，每次父节点选择两个子节点得到的路径和最小值，以此类推
```python
# Modify the original triangle, bottom-up
def minimumTotal(self, triangle):
    if not triangle:
        return 0
    for i in range(len(triangle)-2, -1, -1):
        for j in range(len(triangle[i])):
            triangle[i][j] += min(triangle[i+1][j], triangle[i+1][j+1])
    return triangle[0][0]
```
